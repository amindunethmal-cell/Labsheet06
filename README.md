# Labsheet 06 — Java Swing GUI Forms

A Java desktop application built with **NetBeans** and **Java Swing**, demonstrating GUI form design alongside object-oriented programming concepts. The project contains four independent modules, each pairing a model class with a Swing `JFrame` form.

## Project Structure

```
labsheet06/
├── Labsheet06.java        # Main entry point
├── Book.java              # Book model
├── BookForm.java          # Book GUI form
├── BookForm.form          # NetBeans form descriptor
├── Appointment.java       # Appointment model
├── AppointmentForm.java   # Appointment GUI form
├── AppointmentForm.form   # NetBeans form descriptor
├── Member.java            # Member model
├── MemberForm.java        # Member GUI form
├── MemberForm.form        # NetBeans form descriptor
├── Product.java           # Product model
├── ProductForm.java       # Product GUI form
└── ProductForm.form       # NetBeans form descriptor
```

## Modules

### Book
Stores a book's title, author, and page count. Determines whether the book is a *Short Book* (under 100 pages) or a *Regular Book*.

**Form fields:** Title, Author, Pages  
**Output:** Book type classification and details

### Appointment
Manages medical appointment bookings with fee calculation. Supports appointment types (General, Specialist, Online) and patient categories (Adult, Child, Senior) with corresponding fee adjustments.

**Form fields:** Patient Name, Appointment Type, Patient Category, Base Fee  
**Output:** Final fee after applying surcharges and discounts

### Member
Handles gym/club membership registration. Calculates total membership fees based on membership type (Standard / Premium), number of months, and an optional newsletter subscription add-on.

**Form fields:** Member Name, Membership Type, Number of Months, Newsletter checkbox  
**Output:** Monthly fee, total membership cost

### Product
Tracks product inventory. Calculates the total price for a given quantity and indicates stock status (*Low Stock* if quantity < 5, otherwise *Available*).

**Form fields:** Product Name, Unit Price, Quantity  
**Output:** Total price and stock status

## Requirements

- Java JDK 8 or later
- Apache NetBeans IDE (recommended — `.form` files are NetBeans GUI Builder descriptors)

## Running the Project

1. Open NetBeans and select **File → Open Project**.
2. Navigate to the `labsheet06` folder and open it.
3. Right-click the project in the Projects panel and choose **Run**.
4. The application launches from `Labsheet06.java`. Open individual forms by instantiating and displaying the desired `*Form` class.

## Notes
